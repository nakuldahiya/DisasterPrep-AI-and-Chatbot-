# DisasterPrep AI 🚨

AI-powered Disaster Preparedness Assistant built using Streamlit and Google Gemini API.

## Features
- 💬 Chatbot for emergency guidance
- 📋 Disaster Plan Generator
- 🧠 AI-based responses
- 🌍 Supports multiple disasters

## Tech Stack
- Python
- Streamlit
- Google Gemini API

## How to Run

pip install -r requirements.txt  
streamlit run app.py

## Note
Create a .env file with your Gemini API key:
GEMINI_API_KEY=your_key_here
